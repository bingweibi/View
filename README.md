# View(暂定)
利用空闲时间做的一款App，每天更新知乎日报内容、新闻头条、科技、娱乐、体育、一张图片、音乐播放、视频播放。
还在更新中...

# 主要版块
阅读、福利(美图)、音乐、视频
阅读包括知乎日报、头条新闻、科技、娱乐、体育
